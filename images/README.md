# Screenshots

This folder contains screenshots for documentation purposes.

## Screenshot Files

Please save your screenshots in this folder with the following names:

1. **`control-configuration.png`** ✅ - Control configuration in Maker Portal
   - Shows field selection, label configuration, platform options, and display settings
   - Current file is a placeholder - **please replace with the actual screenshot you just took**

2. **`notification-list.png`** - The notification list view
   - Shows sent notifications with Title, Body, Count, and Recipients columns
   - Displays "New Notification" button and notification cards grouped by title/body

3. **`notification-details.png`** - The notification detail view
   - Shows notification details with recipient names
   - Displays "No recipient assigned" fallback for missing recipients
   - Shows "View recipient names" clickable link

4. **`notification-form.png`** - The notification creation form
   - Shows all recipient selection options (System Users, Teams, Queues, Outlook DLs)
   - Displays Title, Body, Icon Type, and Notification Type fields
   - Shows "Send Notification" button

## How to Add Screenshots

1. Take screenshots of the relevant UI views
2. Save them in this folder with the exact filenames listed above
3. Use PNG format for better quality
4. Ensure no sensitive data is visible in screenshots

These images are referenced in the main README.md file to showcase the control's UI and configuration process.
